Vépárpup Grammar

so first up lets get the general basic sentence structure

Vépárpup technically uses Subject-Object-Verb word order, similarly to languages like Japanese.
however, in practice the word order is generally a bit freeer than that, or at least ambiguous in nature
the main thing to note is that it follows polypersonal agreement, which to explain very loosely means that both the subject and object are marked on the verb itself as affixes - a feature found in many polysynthetic languages.
in vépárpup, the subject and object will often be fused into one singular prefix. for example, when referring to a specific object, one would first state what it is before stating the verb, but following mentions would use ña until if that were to become ambiguous.
here is a list of subject/object prefixes:

vo - 1SG ("i/me/myself")
vó - 2SG ("you/yourself")
ví - 1PL ("we/us/ourselves")
wí - 2PL ("youse/y'all/yous")
ña - 3SG ("he/she/it/that/singular they")
řa - 3PL ("plural they/them")

now for the ones that contain both the subject AND the object:
xřé - x.SUB/itself (such as if you do something to yourself, you would say "vořé(verb)"
vwo - 1sgSUB/2sgOBJ
vwó - 2sgSUB/1sgOBJ
vap - 1sgSUB/3sgOBJ
váp - 2sgSUB/3sgOBJ
ño - 3sgSUB/1sgOBJ
ñó - 3sgSUB/2sgOBJ
xra' - xplSUB/xOBJ
xro' - xSUB/xplOBJ
xrář - xplSUB/xplOBJ

so for a few examples:
"I will eat a fish." - "Vep pávapñon."
"She's going to kill me." - "Páñoípan."
"I can see you." - "Vwoenivri."
"We didn't do that." - "Iñivapra'ñu."
"You hurt your head?" - "Vóřéáñpáíhu?"
